# Submissions Folder

This folder contains all submission CSV files generated by different models and experiments.

## File Organization

All submission files follow the naming convention:
- `my_submission.csv` - Main/latest submission file
- `my_submission_<method>.csv` - Specific method submissions
- `my_submission_weighted_<percentage>.csv` - Weighted ensemble submissions

## Submission Format

All files follow the competition format:
```csv
id,loan_paid_back
593994,0.5234
593995,0.7123
...
```

- Header row required
- `loan_paid_back` must be probabilities (0.0 to 1.0)
- Exactly 254569 data rows (plus header = 254570 total lines)

## Usage

To submit a file, copy it from this folder to your submission location:
```bash
cp submissions/my_submission_weighted_80.csv /path/to/submission/
```

Or use the file directly if the submission system accepts relative paths.

## Verification

Use the verification script to check any submission:
```bash
python verify_submission.py submissions/my_submission_weighted_80.csv
```
